# Инфраструктура

### SLI

* P95
* Uptime
* Количество свободной RAM
* Error rate (5xx ошибки)

### SLO:

* p95 < 3s
* Error rate < 1%
* Uptime за 5 минут не превышает допустимый предел N (N=количество физических ядер)
* RAM > 15%

### Критические уровни (эскалация оператору)
* p95 > 5s
* Error rate > 3%
* Uptime за минуту превышает допустимый предел N, либо Uptime за 5 минут превышает допустимы предел более, чем в 1,15 раз
* Количество свободнной оперативной памяти менее 10%



# Модельный уровень

### SLI:

* ROC-AUC
* TPR 
* TNR
* Prediction confidence distribution

### SLO:

* TNR > 0.92
* TPR > 0.85
* ROC-AUC > 0.90
* Доля низкой уверенности (<0.6) ≤ 15%

### Критические уровни (эскалация оператору)
* TNR < 0.88  модель не допускается к использованию
* ROC-AUC < 0.85  требуется переобучение
* Доля неопределённых предсказаний > 30% - деградация модели


# Бизнес-уровень

### SLI

* Cost savings (экономия на биохимических тестах)

### SLO

* ≥ 20% снижение стоимости предварительного анализа

### Критические уровни (эскалация оператору)

* Экономия < 10% - система не оправдывает внедрение
