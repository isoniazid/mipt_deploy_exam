# 1. use_new_backend_arch

Date: 2026-29-05

## Status

Accepted

## Context

Была предложена новая архитектурная схема для бекенда, для которой предполагалось уменьшение времени отклика. Были проведены тесты на старой и новой архитектуре с замерами временем отклика. Необходимо выбрать, оставаться на старой архитектуре, или перейти на новую

## Decision

- Нулевая гипотеза: В улучшенной системе НЕТ статистически значимого уменьшения задержки
- Альтернативная гипотеза: В улучшенной системе есть статистически значимое снижение задержки

для проверки использовали Т-тест Уэлча. 
Статистически значимыми считали результаты при p<0.05

В результате получилось T-значение: 1875.0793, и р<0.01

Было принято решение перейти на новую архитектурную схему для бекенда

## Consequences

Время отклика снижено в 1,8 раз
